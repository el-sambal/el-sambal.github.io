How to use this LaTeX export?

In order to be able to compile this proof in LaTeX, you should do two things.
First, make sure that you have access to the fitch package by Peter Selinger (https://ctan.org/pkg/fitch), and put on the top of your document:


\usepackage{fitch}


If the fitch package is not installed on your system, you can place the file fitch.sty from https://github.com/OpenLogicProject/fitch/ it in the same directory as the other file(s) you want to compile. If you do this, the \usepackage{fitch} should also work.
